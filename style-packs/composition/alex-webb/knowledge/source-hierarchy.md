# Alex Webb Composition Source Hierarchy

Use sources in this order:

1. Magnum profile pages and Magnum pieces featuring Alex Webb directly
2. Magnum book or body-of-work pages
3. Curated analysis sources that translate the work into practical crop heuristics

If a secondary source over-romanticizes the work, prefer the Magnum definition layer and explain the image through layers, light, color, subject separation, and narrative density.
