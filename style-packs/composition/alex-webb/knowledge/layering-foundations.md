# Alex Webb Layering Foundations

Core visual tests for an Alex Webb-like frame:

- A meaningful foreground, middle ground, and background
- Color blocks or tonal contrast that help separate those layers
- A readable path for light, shadow, or gesture across the frame
- Enough density to feel alive without collapsing into clutter
- Secondary actors or contextual fragments that deepen the story

Cropping should strengthen these relationships, not merely make the frame cleaner.
