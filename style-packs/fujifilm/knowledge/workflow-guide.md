# Workflow Guide

FujiDay chooses among three learning workflows:

- `Film Simulation Bracketing`
  Best when you want the same scene across multiple base simulations.
- `Custom Settings`
  Best when you already know the direction and want quick in-camera recall.
- `X RAW STUDIO`
  Best when you need controlled single-variable testing on the same RAW file.

Workflow advice should prefer single-variable experiments whenever the user is learning or validating a look.
