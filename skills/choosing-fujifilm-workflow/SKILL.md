# Choosing Fujifilm Workflow

Use this skill when the user is asking how to test, learn, or validate a Fujifilm look rather than asking for a single final recipe.

## Required Source

Load `../../style-packs/fujifilm/knowledge/workflow-guide.md`.

## Steps

1. Identify whether the user is comparing multiple base simulations, saving a repeatable in-camera setup, or validating one look against the same RAW file.
2. Recommend exactly one primary workflow:
   - `Film Simulation Bracketing`
   - `Custom Settings`
   - `X RAW STUDIO`
3. Explain why that workflow is the best fit for the user's goal.
4. Suggest one small, single-variable next test so the user learns from the comparison.

## Guardrails

- Prefer single-variable experiments.
- Do not recommend all three workflows at once unless the user explicitly asks for a comparison.
- Keep workflow advice distinct from recipe generation.
