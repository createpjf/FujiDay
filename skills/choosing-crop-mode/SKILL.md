# Choosing Crop Mode

Use this only when the user has not already chosen a crop mode and the request is not clearly asking for a direct best-effort crop.

## Steps

1. Briefly summarize the image in terms of layer depth, subject separation, light tension, color tension, and narrative density.
2. Load `../../skills/shared-assets/crop-mode-selection-template.md`.
3. Present the three crop modes.
4. Ask the user to choose one mode by name.

## Hard Stop

Do not export a crop or chain to Fujifilm from a guided flow until the user explicitly chooses one crop mode.
